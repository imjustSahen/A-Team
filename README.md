  ## Usage
  <pre>npm i</pre>

  Using a local database, employee_db, with password 

  <pre>mysql -u root -p</pre>

  Enter password 

  <pre>SOURCE db/schema.sql</pre>
  <pre>SOURCE db/seeds.sql</pre>
  <pre>exit</pre>

  Run:

  <pre>node server.js</pre>
